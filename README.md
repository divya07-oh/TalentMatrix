# TalentMatrix

TalentMatrix is a high-fidelity, institutional collaboration network for students. It facilitates interdisciplinary innovation through verified node mapping and a proprietary Skill Strength Index (SSI).

## 🚀 Vision
Connecting specialized talent across the academic matrix to foster global innovation through secure, verified collaboration.

## 🛠️ Stack
- **Frontend**: React (Vite), Framer Motion, Lucide Icons, Vanilla CSS.
- **Backend (Skeleton)**: Node.js, Express.
- **Styling**: Premium "Cyber-Institutional" aesthetic (Matrix theme).

## 📱 Modules
- **Landing Page**: Blueprint reveal animations and system thesis.
- **Dashboard**: Real-time SSI visualization and node management.
- **Collaboration Registry**: Dedicated management for project requests.
- **Talent Discovery**: Granular search for specialized skills.
- **Admin Analytics**: Systems oversight and network health.

## 🏁 Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Clone the repository.
2. Navigate to `frontend`:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
3. Navigate to `backend`:
   ```bash
   cd backend
   npm install
   ```
